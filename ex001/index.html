Primeiros comandos

Os primeiros comandos em javascript se assemelham aos primeiros comandos em portugol. são eles:

  • window.alert(), equivale ao escreva, ou seja, mostra uma menssagem na tela para o usuario
  • window.confirm(), mostra uma mensagem na tela e abre um botão de ok e um botão de cancel, é possível armazenar a resposta do usuario em uma váriavel.
  • window.prompt(), faz uma pergunta pro usuario e abre uma caixa para o usuario responder, é semelhante ao leia do visualg.
